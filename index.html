História dos computadores

A história do computador começa com a busca por dispositivos que auxiliem em cálculos, evoluindo de máquinas mecânicas para eletrônicas. Charles Babbage concebeu máquinas analíticas no século XIX, enquanto o ENIAC, criado na década de 1940, foi o primeiro computador eletrônico funcional. A partir daí, houve avanços significativos, como o uso de transistores, circuitos integrados, e a criação dos microprocessadores, levando aos computadores pessoais e à era da computação moderna.

O que é Clients x Servers

Clients:Um cliente é uma entidade que solicita dados ou serviços de um servidor. Pode ser um computador, um smartphone, um programa de software (como um navegador web) ou qualquer outro dispositivo capaz de interagir com um servidor.

Servers:Um servidor é um computador ou programa que fornece serviços ou recursos para outros computadores ou programas (os clientes). Exemplos comuns de servidores incluem servidores web (que hospedam sites), servidores de e-mail (que armazenam e enviam e-mails), e servidores de banco de dados (que armazenam e gerenciam dados).

Um servidor é um computador ou programa que fornece serviços ou recursos para outros computadores ou programas (os clientes). Exemplos comuns de servidores incluem servidores web (que hospedam sites), servidores de e-mail (que armazenam e enviam e-mails), e servidores de banco de dados (que armazenam e gerenciam dados).

Estrutura básica de um HTML

A estrutura básica de um documento HTML consiste em três elementos principais: !DOCTYPE html, html, head e body. O !DOCTYPE html informa ao navegador que o documento é um HTML5. O elemento html engloba todo o conteúdo da página. O head contém informações sobre a página, como título e metadados, enquanto o body contém o conteúdo visível da página.

Listas ordenadas e não ordenadas

Em HTML, listas ordenadas e não ordenadas servem para apresentar itens de forma diferente. Listas ordenadas ( ol ) usam números ou letras para indicar uma ordem específica, enquanto listas não ordenadas ( ul ) utilizam marcadores (como pontos) para apresentar itens sem uma ordem definida.

Listas Ordenadas ( ol ):

  • Ordem Importa: São usadas quando a sequência dos itens é importante e tem significado, como instruções passo a passo ou etapas de um processo
  • Marcadores Numéricos ou Alfabéticos: Por padrão, os itens são numerados, mas podem ser configurados para usar letras ou outros marcadores.
  • Exemplo: "1. Primeiro passo, 2. Segundo passo, 3. Terceiro passo".

Listas Não Ordenadas ( ul ):

  • Ordem Não Importa: São usadas quando a ordem dos itens não é relevante, como uma lista de ingredientes ou tarefas.
  • Marcadores com Símbolos: Tipicamente usam marcadores como pontos, círculos ou quadrados para indicar cada item.
  • Exemplo: "• Item 1, • Item 2, • Item 3".
